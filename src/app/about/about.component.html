About This Project

Project Overview

This is a static website project built with Angular 7, created as a demonstration of building multi-page applications with modern web technologies.

Technologies Used

  • Angular 7: The main framework for building this application
  • TypeScript: For type-safe development
  • Angular Router: For client-side navigation
  • CSS3: For styling and layout

Purpose

This project serves as a foundation for understanding Angular's routing capabilities and component-based architecture. It demonstrates how to structure a simple static website with multiple navigable pages.
